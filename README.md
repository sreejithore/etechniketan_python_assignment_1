1. Write a Python program that prints all the keywords available in Python.
2. Create a variable x = 10 and print its type.
3. Create a tuple with 4 numbers and print its last element and second element.
4. Convert the string "123" into an integer and add 10 to it. Store the result in a variable and
print it.
5. Take a float number and convert it into an integer.
6. Join two strings "Hello" and "World" with a space in between. Find the length of the new
string.
7. Take a boolean variable flag = True and print its type.

9. Given 2 variables:
language = “Python”
version = 3.13
Create a new variable called result with “Python3.13” as the value using the variables
language and version.
10. Accept the following inputs from the user: Student Name, Age, City, Course Name,
Marks in Subject 1, Marks in Subject 2, Marks in Subject 3.
Store all values in appropriate variables and calculate the percent and print it.
11. Create a list named subjects containing: Python, SQL, Excel, Tableau
Perform the following:
a. Display the complete list
b. Display the first subject and the last subject
c. Add one new subject between Python and SQL. Display the updated list
d. Delete Excel from the list and display the updated list
e. Copy the list into another list.
f. Sort the new list in ascending order.
g. Check if Excel is present in the list (use appropriate operator)

attendance = True
assignment_submitted = False
a. Which operator should be applied between these variables to get a True?
b. Which operator should be applied between these variables to get a False?
c. Which operator should be applied to ‘attendance’ to get a False?
